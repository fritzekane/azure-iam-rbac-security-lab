# Azure IAM RBAC Security Lab

This project demonstrates how Role-Based Access Control (RBAC) is implemented in Microsoft Azure to enforce least privilege access.

## Lab Environment

Resource Group:
iam-security-lab

Users Created:

- Cloud Admin
- Developer User
- Security Auditor

## Role Assignments

Cloud Admin → Contributor  
Developer User → Reader  
Security Auditor → Security Reader  

## Security Concepts Demonstrated

- Azure Identity and Access Management (IAM)
- Role-Based Access Control (RBAC)
- Least Privilege
- Separation of Duties
